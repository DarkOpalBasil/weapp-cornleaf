# wechat-weapp-cornLeaf
## 微信小程序-玉米叶片图像上传

### 主要使用以下API：
  * wx.createCameraContext 创建相机上下文
  * wx.chooseImage 选择本地图片文件
  * wx.request 网络请求
  * wx.login 登录
